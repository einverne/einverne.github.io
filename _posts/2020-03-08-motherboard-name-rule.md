主板（Motherboard）作为计算机的基础骨架，其命名往往蕴含了大量关于性能、定位和规格的信息。了解这些命名规则，可以帮助我们在选购时快速筛选出符合需求的产品。

多数厂家遵循一般的规律：**厂商系列 + 芯片组 + 规格后缀**。

例如：`ASUS TUF GAMING B760M-PLUS WIFI D4`

- **厂商系列**: ASUS TUF GAMING (华硕电竞特工系列)
- **芯片组**: B760 (Intel 中端芯片组)
- **规格**: M (M-ATX 紧凑版型)
- **后缀**: WIFI (板载无线网卡), D4 (支持 DDR4 内存)

## 芯片组命名规则 (Chipset)

芯片组决定了主板的基础功能上限，如 CPU 超频支持、PCIe 通道数量、接口扩展性等。芯片组由 CPU 厂商（Intel / AMD）定义。

### Intel 芯片组

Intel 主板芯片组通常由 **字母 + 数字** 组成，字母代表定位，数字代表代数。

- **Z 系列 (如 Z790, Z690)**：高端发烧级。支持 CPU 超频（配合 K 后缀 CPU）和内存超频，提供最丰富的 PCIe 通道和扩展接口。适合发烧友和高性能需求用户。
- **H 系列 (如 H770, H670)**：高端主流级。通常不支持 CPU 超频，但扩展性接近 Z 系列。目前市面上较为少见，因为定位被 B 系列挤压。
- **B 系列 (如 B760, B660)**：主流性价比级。支持内存超频（部分代数开始支持），不支持 CPU 超频。接口数量够用，是大多数普通用户和游戏玩家的首选。
- **H 系列 (入门，如 H610)**：入门级。不支持超频，通道数最少，通常只保留最基础的功能。适合办公机或预算极紧凑的装机。
- **X 系列 (如 X299)**：HEDT（高昂桌面平台）发烧级，对应 i9 Extreme 等至尊系列 CPU，通常用于工作站。

**数字代数说明**：

- **700 系列 (Z790, B760)**: 支持 Intel 12/13/14 代酷睿 CPU (LGA1700 接口)。
- **600 系列 (Z690, B660)**: 支持 Intel 12/13/14 代酷睿 CPU (LGA1700 接口)。
- **500/400 系列**: 对应旧款 10/11 代 CPU。

### AMD 芯片组

AMD 的命名规则与 Intel 非常相似，但在 AM5 时代引入了新的后缀。

- **X 系列 (如 X670, X570)**：高端级。支持 CPU 和内存超频，拥有最多的 PCIe 通道。
- **B 系列 (如 B650, B550)**：主流级。支持 CPU 和内存超频（这是 AMD B 系列的一大优势），扩展性略低于 X 系列，是构建 AMD 平台的性价比首选。
- **A 系列 (如 A620, A520)**：入门级。不支持 CPU 超频，适合办公或轻度娱乐。

**AM5 时代的 "E" 后缀 (Extreme)**：
在 Ryzen 7000 (AM5) 之后，AMD 引入了 `E` 后缀（如 X670E, B650E）。

- **带 E 后缀**: 必须支持 PCIe 5.0 的显卡插槽 **和** M.2 存储插槽。
- **不带 E 后缀**: 通常只要求支持 PCIe 5.0 的 M.2 存储，显卡插槽可能是 PCIe 4.0。

## 四大板厂产品线命名

除了芯片组，各家厂商通过不同的系列名称来区分产品的档次和目标人群。

### ASUS 华硕

华硕的产品线划分非常清晰：

1.  **ROG (Republic of Gamers / 玩家国度)**: 顶级旗舰。
    - **ROG MAXIMUS (Intel) / ROG CROSSHAIR (AMD)**: 纯血 ROG，极致用料，价格昂贵。
    - **ROG STRIX (猛禽)**: 针对主流高端玩家，带 ROG 标，外观酷炫，稍微亲民一些。
2.  **TUF GAMING (电竞特工)**: 中端主流。主打军事风格、耐用性和性价比，是出货量很大的系列。
3.  **ProArt**: 创作者系列。针对设计师优化，通常接口丰富（如雷电口、万兆网卡），外观低调奢华。
4.  **PRIME (大师)**: 入门/商务系列。主打稳定性，功能基础，外观朴素（通常是黑白灰配色）。

### MSI 微星

微星的主板分为四个主要等级：

1.  **MEG (MSI Enthusiast Gaming)**: 旗舰级。
    - 代表型号：**GODLIKE (超神)**, **ACE (战神)**, **UNIFY (暗影)**。极致堆料。
2.  **MPG (MSI Performance Gaming)**: 高端性能级。
    - 代表型号：**CARBON (暗黑)**, **EDGE (刀锋)**, **FORCE (原力)**。主打高性能和 RGB 灯效。
3.  **MAG (MSI Arsenal Gaming)**: 主流军火库级。
    - 代表型号：**TOMAHAWK (战斧导弹)**, **MORTAR (迫击炮)**, **BAZOOKA (火箭筒)**。其中 **迫击炮** 系列是 M-ATX 版型中的明星产品。
4.  **PRO**: 商务办公级。
    - 命名通常直接带 PRO，如 `PRO Z790-A`。主打稳定耐用，性价比高。

### GIGABYTE 技嘉

技嘉目前主要以 AORUS 品牌区分高低端：

1.  **AORUS (大雕)**: 电竞系列，内部又有细分等级。
    - **XTREME / MASTER**: 旗舰级。
    - **ULTRA / PRO**: 高端级。
    - **ELITE (小雕)**: 主流甜点级，性价比高。
2.  **AERO**: 设计师系列。主打银白色外观，针对创作者优化。
3.  **GAMING X**: 中端游戏系列。
4.  **UD (Ultra Durable / 超耐久)**: 入门/主流系列。主打扎实耐用，价格亲民。

### ASRock 华擎

华擎常被称为“妖板之王”，但也拥有正经且细致的产品线：

1.  **Taichi (太极)**: 旗舰系列。齿轮元素设计，用料豪华。
2.  **Phantom Gaming (幻影电竞)**: 游戏系列。从高端到中端都有覆盖（如 PG Velocita, PG Riptide, PG Lightning）。
3.  **Steel Legend (钢铁传奇)**: 主打外观（迷彩/银白）和耐用性，中端市场的热门选择。
4.  **PRO (如 Pro4, Pro RS)**: 主流/商务系列。性价比极高，适合预算有限的用户。
5.  **Aqua**: 限量水冷旗舰，极其昂贵。

## 常见后缀含义

主板型号末尾的后缀通常代表了特定的规格或功能：

- **尺寸相关**:
  - **无后缀**: 通常是标准 **ATX** 大板 (30.5 × 24.4 cm)。扩展性最好。
  - **M**: 代表 **M-ATX** (Micro-ATX) 紧凑型主板 (24.4 × 24.4 cm)。市面上最主流的尺寸，兼容性好，性价比通常最高。
  - **I**: 代表 **Mini-ITX** 迷你主板 (17 × 17 cm)。专为小机箱设计，插槽少但集成度极高，价格通常较贵。
  - **E**: 有时代表 **E-ATX** (Extended ATX)，比标准 ATX 更宽，常见于顶级旗舰板。
- **网络相关**:
  - **WIFI / AC / AX**: 板载无线网卡。`AC` 指 WiFi 5，`AX` 指 WiFi 6/6E，`BE` 指 WiFi 7。
- **内存相关**:
  - **D4**: 明确标识支持 **DDR4** 内存（在 DDR4/DDR5 过渡时期的主板常见）。如果没有标 D4 且是新一代芯片组，通常默认支持 DDR5。
- **其他**:
  - **PLUS / PRO / MAX / V2**: 通常代表同一型号的升级版或改版，供电或接口可能有小幅增强。

## 总结

挑选主板时，可以按照以下步骤解读名称：

1.  **看芯片组**: 确定 CPU 兼容性和是否需要超频（如 `B760` vs `Z790`）。
2.  **看版型**: 确定机箱大小（`M` 代表小板，无后缀通常是大板）。
3.  **看系列**: 确定预算和用料等级（如 `TUF` vs `ROG`, `MORTAR` vs `GODLIKE`）。
4.  **看后缀**: 是否需要 WiFi，确定内存代数（`D4`）。

例如，一块 **MSI MAG B760M MORTAR WIFI II**：

- **MSI MAG**: 微星主流军火库系列。
- **B760**: Intel 中端芯片组。
- **M**: M-ATX 紧凑版型。
- **MORTAR**: 迫击炮型号（该系列中的具体款）。
- **WIFI**: 带无线网卡。
- **II**: 第二代修订版。
