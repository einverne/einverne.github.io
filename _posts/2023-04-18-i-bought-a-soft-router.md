## Proxmox VE

[[Proxmox VE]] 是一个功能强大的开源企业级虚拟化平台。它基于 Debian GNU/Linux，集成了 KVM（基于内核的虚拟机管理程序）和 LXC（Linux 容器），实现虚拟机和容器的统一管理。通过内置的 Web 管理界面，用户可以轻松创建和管理虚拟机、容器、存储、网络和高可用集群，并配备备份恢复、实时迁移、复制、防火墙等多种开箱即用的企业级功能。

Proxmox VE 支持多种存储类型（本地、网络存储如 NFS、iSCSI，分布式存储 Ceph 等）和多种网络配置（桥接、NAT、VLAN、OpenvSwitch），可用于构建超融合基础架构（HCI）。它完全基于软件定义，确保高灵活性和成本效益，适合中小企业、数据中心和实验室使用。用户还可以通过 RESTful API 实现自动化管理。

## iKuai

爱快是一家商业场景网络提供商，拥有 DPI 七层流控、AP 主动勘测技术，有自己的硬件设备，网关、路由、AP 以及交换机等，也有自己路由系统。

iKuai（爱快）是一款基于 Linux 的软路由系统，支持多线负载均衡、流量控制、VPN 接入及防火墙功能。它核心特点是轻量级（仅需 128MB 内存）且高度可定制，适合中小企业和个人用户实现网络管理和优化。

iKuai 还集成了虚拟机功能，允许在支持 x86 硬件平台的路由器上直接运行虚拟机环境，用户可以上传符合要求的软件镜像，在路由器中运行第三方系统或软件，满足多样化需求。iKuai 虚拟机支持多网卡绑定，桥接和 NAT 的灵活网络配置，适合复杂网络环境的部署。

此外，iKuai 提供了开放平台，支持第三方合作伙伴接入设备管理服务，实现高度定制化功能及业务开发。其管理界面简洁易用，支持多级账号权限管理、日志记录和系统健康监测，是一款功能全面且易上手的软路由解决方案.

## pfSense

pfSense 是一款基于 FreeBSD 的开源软路由和防火墙系统，广泛用于网络安全和流量管理。它拥有强大的路由及防火墙功能，支持多 WAN 负载均衡、流量控制、数据包过滤、VPN（IPsec、PPTP）等企业级网络功能。pfSense 配备了用户友好的 Web 管理界面，便于配置和管理网络，同时支持多种软件包扩展以增强功能，如入侵检测和漏洞扫描等。

pfSense 起源于 m0n0wall 项目，经过改进增加了更多功能，适合全天候运行的稳定网络环境。它支持 802.1Q VLAN 标准，方便进行复杂网络分段与管理，被很多企业和网络爱好者用于构建安全、高效的网络系统。pfSense 既可以运行在专用硬件上，也能虚拟化部署，灵活性强且免费开源，没有许可限制，且提供商业支持和硬件选项。

## OpenWrt

OpenWrt 是一个基于 Linux 的开源嵌入式操作系统，专为智能路由器和各种网络设备设计。它不是固定不可更改的固件，而是提供了完全可写的文件系统和强大的软件包管理系统，使用户能够根据需求定制和扩展路由器功能。OpenWrt 支持多种硬件平台，包括常见的路由器芯片组，具备高度模块化和灵活性。

OpenWrt 的主要优势包括灵活的定制化能力、大量可用的软件包、强大的网络功能和安全性（如防火墙、VPN、SSH），以及活跃的开源社区支持。用户不仅可以使用图形界面进行基本配置，还能通过命令行进行深度定制。它还支持无线网络管理、带宽控制、访问控制和各种智能家居集成，适合开发者和高级用户打造高性能的网络环境。

我安装的 OpenWrt 版本是 eSir 维护的版本。

## LEDE

LEDE（Linux Embedded Development Environment）是一个基于 OpenWrt 的开源路由器操作系统项目，最初是 OpenWrt 的一个分支，目的是提高系统的稳定性、安全性和性能。LEDE 具有超过 3000 个标准化软件包，支持广泛的硬件设备，包括许多旧型号的路由器。它强调高安全标准，如默认禁用 Wi-Fi、无弱密码和无后门程序，系统组件定期更新以及时修复漏洞。

LEDE 的设计注重性能和稳定性，所有支持设备的固件组件都经过广泛测试，并拥有强大的社区支持。用户可以通过图形界面或命令行进行灵活配置和管理，支持丰富的网络功能如防火墙、NAT、QoS 和 VPN。2018 年，LEDE 项目与 OpenWrt 合并，合而为一，继续作为 OpenWrt 的一部分进行开发和维护，成为目前广泛使用的路由器开源系统基础。
